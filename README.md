# 📋 My Python Full-Stack To-Do App

यह पाइथन (Flask) और SQLite डेटाबेस से बनी एक पूरी तरह काम करने वाली **Full-Stack Web Application** है। इसमें यूज़र अपने रोज़ के काम जोड़ सकता है, पूरे होने पर उन्हें टिक (Strikethrough) कर सकता है और डिलीट भी कर सकता है।

## ✨ फीचर्स (Features)
- 📝 **Task Addition**: नए काम जोड़ें जो सीधे डेटाबेस में सेव होते हैं।
- ✅ **Checkbox & Line-through**: काम पूरा होने पर टिक करें और कटा हुआ इफ़ेक्ट देखें।
- ❌ **Task Deletion**: पूरे हो चुके कामों को लिस्ट से हमेशा के लिए हटाएँ।
- 💾 **SQLite Database**: ऐप बंद होने पर भी आपका डेटा सुरक्षित रहता है।

## 🚀 कंप्यूटर पर कैसे चलाएँ (How to Run)
1. प्रोजेक्ट फ़ोल्डर में टर्मिनल खोलें।
2. यह कमांड चलाकर सर्वर शुरू करें:
   ```bash
   python main.py
   ```
3. ब्राउज़र में `http://127.0.0.1:5000` पर जाएँ।
